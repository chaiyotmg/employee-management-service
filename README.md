# Employee Management System (MVC)

## Tech Stack

- **Backend**: Node.js, Express.js
- **Database**: SQLite3 (for local development only)
- **Template Engine**: EJS
- **Styling**: Tailwind CSS

## Prerequisites

- [Node.js] (v22.22.0)

## Getting Started

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Setup Database (Migration & Seeding)**
   ```bash
   npm run migrate
   ```

3. **Run the Application**
   ```bash
   npm run start
   ```

4. **Access the App**
   Navigate to `http://localhost:3000` in your web browser.
